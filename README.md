# Payroll Projection Engine
AI-assisted workforce cost forecasting tool built with Python + Streamlit.

**Live App:** https://payroll-projection-engine-fttuzbtszxcpkavpmia2jj.streamlit.app

Built by [The Finance Forge](https://thefinanceforge.com)

## Features
- 12-month payroll projections with benefits and tax modeling
- 3 increment models: Anniversary, Fiscal Year Full, Fiscal Year Prorated
- Department budget tracking with variance analysis
- What-If raise sensitivity across 3 scenarios
- Attrition impact modeling
- Excel export
